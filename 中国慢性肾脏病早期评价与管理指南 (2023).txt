📘 【中国慢性肾脏病早期评价与管理指南 (2023) - 核心法则】
(Agent 系统提示词专用 - 针对中国本土临床环境优化)

1. 诊断、分期与危险分层

诊断标准：肾损伤标志（UACR ≥ 30 mg/g、尿沉渣异常、电解质异常、组织学/影像学异常、肾移植史）和/或 GFR < 60 ml/min/1.73m²，持续时间 > 3个月 。


分期标准：沿用 G1-G5 分期 。特别强调对 eGFR 为 45~59 ml/min/1.73m² 且无其他肾损证据的老年人，建议加测胱抑素 C 以明确诊断，避免过度诊断 。


危险分层 (表3)：根据 GFR 分期和尿白蛋白分级，将风险分为：低危、中危、高危、极高危 。

2. 筛查与预防

高危人群：推荐对糖尿病、高血压、心血管疾病、老年群体进行定期筛查 。


频率与内容：每年至少一次 。内容包括尿白蛋白和基于血肌酐的 eGFR；基层若无定量条件可先查尿常规 。


一级预防：推荐 SGLT2 抑制剂用于糖尿病合并动脉粥样硬化性心血管疾病患者的 CKD 一级预防 。

3. 重要控制目标 (本土化数值)
血压控制：


非透析患者：建议 < 130/80 mmHg；若耐受，收缩压可进一步控至 < 120 mmHg 。


老年患者：建议 < 140/80 mmHg 。


血糖控制 (HbA1c)：目标应个体化。健康情况良好者 < 6.5%；有严重低血糖史、预期寿命短或合并严重并发症者 < 8.0% 。


血脂控制 (LDL-C)：高危人群 < 1.8 mmol/L；极高危人群 < 1.4 mmol/L 。

4. 药物治疗路径 (核心决策)
SGLT2 抑制剂 (达格列净/恩格列净)：

推荐用于 CKD 合并或不合并糖尿病患者 。


启动阈值：eGFR ≥ 20 ml/min/1.73m² 时启动，可持续应用至透析或肾移植前 。

ACEI/ARB：


A3 期：无论是否合并高血压或糖尿病，均推荐使用 。


A2 期：合并糖尿病或高血压时推荐使用 。


禁忌：严禁 ACEI 和 ARB 联合使用 。

非甾体 MRA (非奈利酮)：

用于糖尿病合并蛋白尿患者，在已用足量 ACEI/ARB 和/或 SGLT2i 基础上加用，优先考虑非甾体类 。

糖尿病用药：


二甲双胍：建议用于 eGFR ≥ 30 的 CKD 合并 T2DM 患者 。


SGLT2i：推荐用于 eGFR > 25 的 CKD 合并 T2DM 患者 。


GLP-1RA：前两者血糖不达标或不能使用时推荐 。

5. 生活方式与并发症管理
饮食：


低钠：钠摄入应低于 2.3 g/d (约食盐 < 5.8 g) 。


蛋白：CKD 1~2期控制在 0.8~1.0 g/(kg·d)；3~5期代谢稳定者限制摄入，以降低 ESKD 风险 。


运动：推荐进行合理、适度、规律的运动 。


贫血：推荐使用 ESAs、铁剂或 HIF-PHI (如罗沙司他) 。


高钾血症：推荐使用环硅酸锆钠或阳离子交换树脂进行治疗 。

6. 三级管理体系

联动模式：建立“社区卫生服务中心-区属医院-市级医院”的三级联动体系 。


职能分工：社区负责长期监测管理；区/市级负责疑难、危重病诊治 。
